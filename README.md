# UTS-GRAFKOM
Judul : Mini Sacene Rumah Masa depan Grafika 2D 

menggunakan 4 Algoritma 
1. Algoritma DDA
2. Algoritma Gambar Lingkaran (Midpoint Circle)
3. Algoritma Gambar Poligon
4. Transformasi Geometris 2D (translasi, rotasi, skala,
refleksi)

Aplikasi ini merupakan mini scene grafika 2D yang dibuat menggunakan bahasa pemrograman Python dengan pustaka Tkinter sebagai media kanvas. Aplikasi menampilkan sebuah lingkungan sederhana yang terdiri dari rumah dua lantai, jalan raya, pepohonan, matahari dengan efek cahaya, dan awan. Seluruh objek digambar menggunakan algoritma grafika komputer manual, tanpa menggunakan fungsi gambar siap pakai seperti create_rectangle atau create_oval.
Aplikasi ini bersifat interaktif, di mana usser dapat melakukan transformasi geometris terhadap objek rumah menggunakan input keyboard, sehingga dapat mengamati secara langsung efek translasi, rotasi, skala, dan refleksi dalam grafika 2D.
cara menjalankan mini scene rumah masa depan ini saya meninputkan beberapa huruf untuk melakukan tranformasi terhadap rumah 
Tombol	Fungsi
← ↑ → ↓	Menggeser rumah (translasi)
Q	Rotasi ke kiri
E	Rotasi ke kanan
W	Perbesar rumah
S	Perkecil rumah
X	Refleksi sumbu X
Y	Refleksi sumbu Y
R	Reset posisi
ESC	Keluar program
